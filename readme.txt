A simple WooCommerce addon plugin to get notification on different platform on WooCommerce action.

== Description ==
A simple WooCommerce addon plugin to get notification on different platform on WooCommerce action. For example, you can get notification on Slack when a new order is placed. Or you can get notification on Discord when a new order is placed. Or you can get notification on both Slack and Discord when a new order is placed. There are many other notification options.

For documentation, please visit the plugin's homepage at https://devrafi.com/platform-notification-for-woocommerce/
